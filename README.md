# B2B Marketplace

Next.js 16 marketplace frontend with signup, login, OTP verification, onboarding, and dashboard pages.

## Run Locally

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Test Accounts

The local mock API includes these verified demo users:

```text
buyer@example.com / password123
supplier@example.com / password123
```

New signup OTPs use the local demo code shown on the OTP page.

## Scripts

```bash
npm run dev
npm run lint
npm run build
```
